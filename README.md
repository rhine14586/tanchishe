# 赛博朋克风格贪吃蛇游戏

一个具有赛博朋克视觉风格的HTML5贪吃蛇游戏。游戏具有霓虹灯效果、扫描线动画和复古像素风格的界面设计。

## 游戏特点

- 赛博朋克视觉风格
- 霓虹灯光效果
- 流畅的游戏控制
- 实时分数显示
- 响应式设计

## 如何游玩

使用键盘方向键控制蛇的移动方向：
- ↑ 向上移动
- ↓ 向下移动
- ← 向左移动
- → 向右移动
